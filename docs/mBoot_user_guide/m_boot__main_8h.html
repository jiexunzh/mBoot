◆ mBoot_run_app()

uint8_t mBoot_run_app ( void )

如果有效，跳转到主应用程序

返回值
1表示成功，0表示APP无效或跳转失败
注解
APP的中断向量表： ApplicationAddress: APP的堆栈指针的地址，默认的堆栈指针是主堆栈指针 MSP ApplicationAddress + 4: APP的复位向量地址，指向复位中断向量Reset_Handler 运行Reset_Handler只是软件复位，不会触发真正的硬件复位（如复位所有寄存器、重新启动外设等）
