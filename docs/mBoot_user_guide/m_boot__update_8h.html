详细描述

mBoot 固件更新模块头文件

声明了固件更新过程的错误码枚举类型与主处理函数接口。 该模块实现了基于串口的 IAP 升级机制。

枚举类型说明

◆ update_error_t

enum update_error_t

固件更新过程中的错误码定义

枚举值
BINSIZE_TIMEOUT 

固件大小字段接收超时

BINSIZE_STRING_ERROR 

固件大小字段格式错误（未以 BINSIZE= 开头）

BINSIZE_VALUE_ERROR 

固件大小字段值无法解析

PACKET_TIMEOUT 

固件数据包接收超时

PACKET_HEADER_ERROR 

包头或包序号错误

CRC_CCITT_ERROR 

CRC 校验错误，超过最大允许次数

函数说明

◆ firmware_update_process()

int32_t firmware_update_process ( void )

执行一次固件更新流程

注解
包括应答上位机、接收文件大小、循环接收固件包、写入 Flash 等操作
返回值
正值：BIN文件总字节数；负值：update_error_t 中定义的错误码
