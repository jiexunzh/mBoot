详细描述

mBoot 硬件抽象层接口头文件

声明 mBoot 所依赖的所有硬件相关接口函数，包含：

  • UART 初始化与收发
  • Flash 读写
  • 微秒级延时
  • 1KHz 定时器
  • RS485 控制

所有接口需要用户根据目标平台实现，便于移植。
