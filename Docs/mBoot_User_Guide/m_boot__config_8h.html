详细描述

mBoot 配置文件：定义硬件接口宏、功能开关、地址分区与命令标志

本文件用于配置 mBoot 启动加载器与应用程序共用的关键参数， 包括是否使用监听更新命令、使用 RS485、地址分区、串口命令标志字段等。

所有配置项应根据实际硬件平台进行适配。

注解
"mBoot_bootloader" 和 "mBoot_app" 均需包含本头文件以统一参数。
